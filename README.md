# Resume-to-Job-Automation-Engine
This automation is an end-to-end AI-driven workflow that enables candidates to upload their resume and receive personalized job recommendations via email.

The system intelligently extracts structured information from the resume, fetches relevant job listings from an external job API, ranks them using a Large Language Model (Gemini), and delivers the top matching opportunities directly to the candidate’s inbox.
